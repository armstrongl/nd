# Code of conduct

1. Participants will be tolerant of opposing views.
2. Participants must ensure their language and actions are free of personal attacks and derogatory remarks.
3. When interpreting the words and actions of others, assume good intentions.
4. Behaviour which can be reasonably considered harassment will not be tolerated.

## Reporting

To report a violation, contact the project maintainers privately via a [GitHub security advisory](https://github.com/armstrongl/nd/security/advisories/new).
